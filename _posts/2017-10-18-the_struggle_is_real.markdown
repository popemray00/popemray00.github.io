  Flatiron has been a blessing in my life, Im so happy to be a student here. I'm gathering so much valuable information that will help me in the near future. The one struggle I'm having though is that I load so much into my brain at once that its hard to retain all the info I learned and I keep having to go back to previous lessons. Now I'm sure it happens to most people, but for me it happens more once I finish for the day and come back the next day. 
	
	I've done some reading on the web on ways to help retain information and found that going for a run before you start your day or start learning will help. I have started doing this on a regular basis and have found it working great, I seem to be focusing more and when I come back the next day I'm remembering more then I used to. I would highly recommend a little run before you start working if you'r having the same issues. I'm getting more work done and am even more motivated to keep going

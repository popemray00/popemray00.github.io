Going into React and Redux, I had no idea what I was getting into except, everything I've heard about it was positive opinions and news. So I was excited to jump in and see what this was all about. I always start off by just scanning the list of lessons to see if there will be any concepts I've already learned in another language, also to take a mental note on the requirements I'll need to meet on the React project. As i scanned I noticed there are a few things I have seen before like forms, events and fetch. And then there was props, state, dispatch, map state to props and many other things that were the unknown, which got my brain ready to get to work. Everyday as the kids went off to school I would get on my laptop and start my React journey, I don't really know what happened those days outside of me learning the time would just fly by and when I looked up it was already time to go pick the kids up, I was really into this React stuff, I was actually picking it up pretty well I felt like I was becoming a developer. I cruised right through React and Redux, it blew me away how fast I was picking it all up and now that I'm at the end I'm even more excited to get into a job hopefully working with React.

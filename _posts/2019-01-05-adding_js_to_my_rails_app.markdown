Before I started my coursework for the rails & javascript, I decided to try a new approach which was to look ahead at the requirements for the project. I figured this new strategy would allow me to have an idea of what I’ll be implementing into my app and I could make snippets of concepts that will help with the requirements later on. 

Even with notes, snippets and everything else I had written down, I had a real struggle with getting certain features to work the way I wanted. For example, trying to display the correct JSON data to the DOM via a AJAX get request had my head going in circles until I had google searched it enough that I was able to build the right function that would allow me to do exactly what I wanted. 

I had more success with the requirement for the AJAX post request and building a prototype to go along with my JavaScript model object. The process seemed to be clear and I was able to get the post request up and running in just a few hours.

I’ve really enjoyed working with both Rails and JavaScript and plan to keep learning these two languages together as a further my career.
